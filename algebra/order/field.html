Linear ordered fields #

A linear ordered field is a field equipped with a linear order such that

  • addition respects the order: a ≤ b → c + a ≤ c + b;
  • multiplication of positives is positive: 0 < a → 0 < b → 0 < a * b;
  • 0 < 1.
